DOM Manipulation

Todays topic is about DOM Manipulation and its attributes.

  • Get attributes

    • In get attributes we will get to know about the value of id or class what we have assigned

  • Set attributes

    • In set attributes we can set the attributes as per our requirements

  • Inserting Elements

    • There are two steps to insert Elements that is create and add.

  • Append

    • Adds at the end of the node(inside)

  • prepend

    • Adds at the start of the node(inside)

  • before

    • Adds before the node(outside)

  • after

    • Adds after the node(outside)
